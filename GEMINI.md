# React + TypeScript + Vite (TSX)

- Bahasa: TypeScript (strict mode aktif).
- Ekstensi file: `.tsx` untuk komponen React, `.ts` untuk utils/hooks/konfigurasi.
- Struktur Komponen: Gunakan `interface` atau `type` untuk mendefinisikan Props.
- Path Alias: Gunakan `@/` untuk mengakses folder `src/` (sudah diset di vite.config.ts).
- React Versi: 18+ dengan Vite.
- Aturan Kode: Hindari `any` sebisa mungkin. Lebih suka fungsi biasa dengan return type `JSX.Element`.
- Environment: Variabel lingkungan ada di `.env` dan bertipe string.
- Styling: (isi sesuai preferensimu, misal Tailwind / CSS Modules / Styled Components).
